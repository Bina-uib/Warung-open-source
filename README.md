# OpenResto POS - Sistem Kasir Restoran

OpenResto POS adalah aplikasi berbasis web yang dibangun menggunakan **Laravel** untuk membantu operasional restoran, mulai dari manajemen menu, pengelolaan pesanan pelanggan, hingga dashboard laporan penjualan.

## Fitur Utama
- Manajemen Menu: Tambah, edit, dan hapus menu makanan/minuman dengan kategori.
- Transaksi (Point of Sale): Pemrosesan pesanan pelanggan dengan input nomor meja dan jenis pesanan (*Dine-in/Takeaway).
- Manajemen Meja: Pemantauan status ketersediaan meja restoran.
- Dashboard & Laporan: Visualisasi omset dan riwayat transaksi.
- Cetak Struk: Output rincian pembayaran otomatis.

## Persyaratan Sistem
Pastikan lingkungan lokal Anda sudah terinstall:
- PHP >= 8.2
- Composer
- MariaDB / MySQL
- Node.js & NPM
